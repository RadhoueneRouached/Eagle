Eagle Game
=====

Eagle's Wings is a game for children whose age is between 5 and 10 years, the game is composed of 4 stages:

* 1st stage: Puzzle
* 2nd stage: Coloring
* 3rd course: life cycle of the eagle
* 4th stage: quizzes
